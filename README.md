# Verbum

## Propósito

Este projeto nasceu da necessidade de modernizar a infraestrutura de serviços da biblioteca escolar. Atualmente, estudantes e servidores enfrentam um processo de consulta e empréstimo demorado, pouco acessível e analógico.

O problema: O sistema atual carece de ferramentas digitais integradas e de uma interface limpa e moderna.

A solução: Desenvolver uma plataforma centralizada, prática e intuitiva que permita aos usuários:

* Consultar o acervo em tempo real.

* Realizar reservas de títulos de forma remota.

* Renovar empréstimos com apenas alguns cliques.

* Gerenciar pendências e multas remotamente.

## Tecnologias Utilizadas

O projeto foi construído utilizando um ecossistema moderno para garantir portabilidade entre web e dispositivos móveis, além de uma infraestrutura de dados escalável.

### Desenvolvimento Web

* HTML5 & CSS3: Estruturação e estilização da interface responsiva.

* JavaScript: Implementação de comportamentos dinâmicos e integração com APIs.

* PHP: Processamento lógico no lado do servidor e comunicação com o backend.

### Desenvolvimento Mobile

* Kotlin: Linguagem oficial para o desenvolvimento nativo Android, focando em performance e segurança.

* XML: Definição de layouts e interfaces do aplicativo móvel.

### Infraestrutura e Dados

Firebase (Google): Utilizado como Sistema de Gerenciamento de Banco de Dados (SGBD) NoSQL, além de prover autenticação em tempo real e hospedagem.

### Ferramentas de Design e Código

* Figma: Criação de protótipos de alta fidelidade e design de experiência do usuário (UX/UI).

* VS Code: IDE principal para o desenvolvimento do módulo Web.

* Android Studio: IDE oficial para o desenvolvimento do módulo Mobile.

### Gestão e Controle

* GitHub: Sistema de Controle de Versão (Git) para colaboração e armazenamento do código-fonte.

* Trello & GitHub Projects: Utilizados para a gestão ágil do projeto (Kanban), organização de tarefas e cronograma.
